Customer Churn Prediction Model using Machine Learning to identify customers who are likely to leave a service based on customer behavior, 
tenure, monthly charges, and contract details. 
The project includes data preprocessing, visualization, feature engineering, and predictive modeling.



 Customer Churn Prediction

This project predicts customer churn using Machine Learning techniques.
The model analyzes customer data and identifies customers who are at high risk of leaving the company.

 🔹 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Model Training
* Churn Prediction & Insights

🔹 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

🔹 Key Insights

* Customers with low tenure and high monthly charges are more likely to churn.
* Long-term contract customers have lower churn rates.

 🔹 Goal

To help businesses improve customer retention by identifying high-risk customers early.
